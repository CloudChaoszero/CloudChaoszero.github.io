SQL Code Demo

1. Get The name from candidate that has the most votes

SELECT Name FROM Candidate WHERE Candidate.`id` = (SELECT CandidateId FROM VOTE GROUP BY CandidateId ORDER BY COUNT(Vote.`Id`) DESC LIMIT 1)

2. Write code to provide a list of distinct clients that have an email address, signed up for BI after 01/01/2010, and purchased mascara after 3/10/2010

SELECT DISTINCT(clean_customer.customer_id), clean_customer.email_address, clean_customer.BI_signup_date,ls.order_date, p.class FROM (SELECT * FROM customer WHERE BI_signup_date != "not BI") clean_customer INNER JOIN line_sales ls ON clean_customer.customer_id = ls.customer_id INNER JOIN product p ON ls.product_key = p.product_key WHERE (clean_customer.email_address != "" OR clean_customer.email_address IS NOT NULL) AND clean_customer.BI_signup_date > "01/01/2010" AND (p.class = "mascara" AND ls.order_date > "3/10/2010");
